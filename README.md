# 🚀 iOS Algorithms & Data Structures in Swift

A curated library of LeetCode and HackerRank solutions in modern Swift, built for continuous mastery and performance benchmarking.

This repository serves as a personal reference vault and a demonstration of writing clean, highly performant, and well-tested code for iOS development.

## 🎯 Goals & Principles

- **Modern Swift:** Implementations utilizing Swift 5/6 features, higher-order functions, and modern concurrency where applicable.
- **Test-Driven Development (TDD):** Every algorithmic solution is backed by `XCTest` cases to ensure reliability and handle edge cases gracefully.
- **Time & Space Complexity:** Focus on optimal $O(N)$ and $O(1)$ solutions, analyzing Big O notation for every single problem.

## 📂 Repository Structure

The algorithms are categorized strictly by the optimal pattern required to solve them, following industry-standard technical interview roadmaps:

- `/ArraysAndHashing` (Frequency counters, Hash Maps, Sets)


## 🛠️ Tech Stack

- **Language:** Swift
- **Framework:** XCTest
- **Environment:** Xcode / macOS
